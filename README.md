# acezravel-api
> My simple repository learn API with Laravel.

## Endpoints

- #### List all articles.
```
GET /api/articles
```

- #### Get an article.
```
GET /api/article/{id}
```

- #### Add an article.
```
POST /api/article
```

- #### Update an article.
```
PATCH /api/article/{id}
```

- #### Delete an article.
```
DELETE /api/article/{id}
```
